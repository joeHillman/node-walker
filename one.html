Page 1
The idea is to step thru a list of menu options, while only seeing the prior and next menu items. I needed an effective way to replicate stepping thru this list, but working with an existing menu, that could be updated at anytime. Current page is flagged at the top and a three up option is provided on the left to display the current page you are on in the menu. A javascript free fallback is provided (go ahead and call it old skool), to render the menu usable in a javascript free environment.
